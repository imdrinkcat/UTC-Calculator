# UTC-Calculator UTC累计秒计算器
## 功能描述：
计算累计秒的小工具，默认为计算“2009年1月1日 00:00:00 UTC+00”至输入日期（北京时间 UTC+08）的秒数。

## 使用说明
只包含一个html文件，使用浏览器打开即可使用。推荐使用Chrome浏览器，Safari浏览器无法输入秒数。

工具已参考其他的时间戳工具进行了验证，结果是可靠的。
